# Changelog Summary

This changelog documents updates to shadcn-svelte, a component library for Svelte applications.

## Recent Highlights

**October 2025** introduced seven new utility components addressing common development needs: Spinner for loading states, Kbd for keyboard key display, Button Group for action grouping, Input Group for enhanced inputs, Field for form management, Item for list displays, and Empty for blank states.

**June 2025** overhauled calendar functionality with "completely overhauled the `Calendar` and `RangeCalendar` components to support dropdowns for the month and year selectors" plus 30+ calendar building blocks.

**May 2025** brought Tailwind v4 support and introduced Charts as preview components, alongside custom registry functionality enabling developers to "publish your own components and share them with the community."

## Notable Earlier Updates

- **March 2024**: Introduced Blocks (pre-built, ready-to-use components) and added Breadcrumb and Scroll Area components
- **February 2024**: Added Resizable component and optimized icon imports for better dev server performance
- **January 2024**: Four new components launched: Carousel, Drawer, Sonner, and Pagination
- **December 2023**: Calendar, Range Calendar, and Date Picker components added
- **October 2023**: Command and Combobox components introduced with formsnap updates

The library emphasizes accessibility, responsiveness, and composability throughout its component ecosystem.
